# `useCurrent`

The `useCurrent` hook manages the state of the current question and category in the application. It provides state values and functions to update these states, with automatic inference of the category based on the current question.

## Usage

```typescript
import useCurrent from '@/publicodes-state/hooks/formProvider/useCurrent'

function MyComponent() {
  const {
    currentQuestion,
    currentCategory,
    setCurrentQuestion,
    setCurrentCategory,
  } = useCurrent()

  // Example usage
  setCurrentQuestion('New question')
  console.log(currentCategory) // Logs the inferred category
}
```

## API

### Returned States

- **`currentQuestion`**: The current question (`string | null`).
- **`currentCategory`**: The current category (`string | null`).

### Returned Functions

- **`setCurrentQuestion(newCurrentQuestion: string | null)`**: Updates the current question and automatically infers the corresponding category.
- **`setCurrentCategory(newCurrentCategory: string | null)`**: Updates the current category.
