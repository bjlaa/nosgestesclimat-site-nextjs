# `useCurrentSimulation`

## Purpose

`useCurrentSimulation` is a custom React hook that manages and provides access to the current simulation state within your application. It offers functionality to both retrieve the simulation data and update it.

## Usage

### Importing the Hook

Import the `useCurrentSimulation` hook into your component:

```typescript
import useCurrentSimulation from '../path/to/useCurrentSimulation'
```

### Using the Hook

Invoke the `useCurrentSimulation` hook inside your functional component to access simulation data and update methods:

```typescript
function MyComponent() {
  const {
    id,
    date,
    situation,
    foldedSteps,
    actionChoices,
    persona,
    computedResults,
    progression,
    defaultAdditionalQuestionsAnswers,
    polls,
    groups,
    savedViaEmail,
    update,
    updateCurrentSimulation,
  } = useCurrentSimulation()

  // Use the simulation data and update functions as needed
}
```

### Returned Values

The hook returns an object with the following properties:

- **Simulation Data**: Various properties related to the current simulation, such as `id`, `date`, `situation`, and more.
- **`update`**: A function to update the current simulation.
- **`updateCurrentSimulation`**: A more granular function to update the simulation, allowing detailed control over the update process.

In this example, `SimulationComponent` uses `useCurrentSimulation` to display the current simulation situation and provides a button to update it.
